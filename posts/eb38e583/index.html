Linux学习-入门篇 (三)

发表于2024-02-14|更新于2024-03-16|技术
|字数总计:1.1k|阅读时长:3分钟|阅读量:

认识命令行

  • 命令行（Command Line）是一种与计算机交互的方式，它允许用户通过在终端输入文本命令来控制和操作计算机。
  • 在命令行中，每个命令都由一个关键字（或称命令名）和一系列参数组成。
  • 命令行实际上就是一个电脑控制的终端，让你可以通过输入命令来操控计算机。

Tips：想象一下，这就像是一个神秘的组织，你通过特定的密码或者暗号（也就是命令），就能解锁并获得计算机的力量（让计算机执行某些操作）。

认识Shell

  • shell是一个用 C 语言编写的程序，它是用户使用 Linux 的桥梁。用户通过Shell访问操作系统内核的服务。
  • Shell可以理解为一个命令翻译器，它位于操作系统和计算机用户之间，负责接收用户的指令，然后将这些指令转换成计算机能理解的命令，从而让计算机执行相应操作。

Tips：我们可以把shell想象成一个翻译员，他能理解用户说的话，然后帮用户与计算机沟通。

认识Bash

  • Bash可以理解为一个更加智能、高效的终端程序。它不仅可以执行简单的命令，还支持复杂的脚本编写，允许用户通过编写脚本实现自动化操作。
  • Bash是Linux系统和Unix系统的一种Shell（命令行环境），是目前绝大多数Linux发行版的默认终端程序。

Tips：Shell与Bash是包含与被包含的关系。
举例来说，在社会中有翻译官这个职业，它是由许多从业者共同组成的职业名称，而Bash则是其中一个出色的成员，是Shell终端程序中的一份子。

命令详解

常见的执行Linux命令的格式是下面这样的：
root@localhost:~# 命令 [参数] [对象]

:和@ 这是提示符的分隔符号，没有特殊含义。
root 显示的是系统当前登录的用户。
localhost 当前登录的主机名
~ 代表用户当前所在的目录
ps: 之后会讲目录的知识
# 命令提示符
超级用户是 #
普通用户是 $
ps：之后会讲用户的知识
命令 命令的名称或命令的关键字。
可以理解为语法中的动词。
比如 穿衣服中的穿。
参数 命令的执行要求
可以理解为语法中的修饰词。
用于对命令进行调整，让调整过的命令能更好地贴合工作需求。
比如 裁剪衣物，让它们更加贴合你的体型。
对象 命令的执行对象
可以理解为语法中的名词。
一般指要处理的文件、目录、用户等资源名称，也就是命令执行后的“承受方”。
比如 穿衣服中的衣服。

第一个命令 man

Linux命令具有这参数多且格式复杂，因此，就有Linux初学者问“Linux系统中有那么多命令，我怎么知道某个命令是干嘛用的？在日常工作中遇到了一个不熟悉的Linux命令，我又怎样才能知道它有哪些可用参数呢？”man命令就可以帮助到你，man 后面直接接其他命令名称可以查看其他命令的帮助

  • man命令有很多参数
  • 其中分为短格式和长格式。
    |短格式|长格式|
    |:----------😐:----------😐
    |man [-h]|man [--help]|
  • 注意到了吗？ man的命令带了[]，这说明我们可以直接执行man命令

退出请使用 q 点一下键盘的q键

  • OK 恭喜你，你已经学会了Linux入门的第一个命令 man

快捷键

学会快捷键能起到事半功倍的效果

  • Ctrl + C：结束（终止）当前命令
    • 当同时按下键盘上的Ctrl和字母C的时候，意味着终止当前进程的运行。
    • 假如执行了一个错误命令，或者是执行某个命令后迟迟无法结束，这时就可以按下Ctrl+C组合键，就会立刻终止此命令。
  • Tab：实现命令的自动补全
    • 当你输入命令、文件或目录的前几个字符时，它会自动帮你补全。
  • Ctrl + L：清屏
    • 清理屏幕上的日志等信息。
