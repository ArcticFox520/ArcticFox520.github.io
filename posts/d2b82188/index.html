Java程序开发三步骤：编写、编译、运行。

  • 将 Java 代码编写到扩展名为 .java 的源文件中。
  • 通过 javac 命令对该 java 文件进行编译，生成一个或多个字节码文件。
  • 通过 java 命令对生成的 class 文件进行运行。
