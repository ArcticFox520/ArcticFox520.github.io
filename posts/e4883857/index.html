Linux学习-入门篇 (一)

发表于2024-02-13|更新于2024-03-09|技术
|字数总计:1k|阅读时长:3分钟|阅读量:

> 在正式开始学习之前，我先唠唠嗑，因为有一个非常现实的问题——学习是一件非常痛苦的事情，如果说学习Linux系统真的很简单，那必定是骗子的谎言。在每天起床后的几分钟时间里，大脑都会陷入斗争状态：是该聊会儿天呢，还是要追个美剧呢，还是学习一下那该死的Linux系统呢？这个时候，请不要忘记自己的初心，未来的你一定会感谢现在努力的自己。

Linux概述

Linux内核最初只是由芬兰人林纳斯·托瓦兹（Linus Torvalds）在赫尔辛基大学上学时出于个人爱好而编写的，当时还在读大学的Linus想要基于Unix的原则和设计来创建一个免费的开源系统，从而代替MINIX操作系统。

  1. Linux基于GNU 通用公共许可证（GPL）的开放源代码特性，迅速得到了GNU源代码开放计划和一大批程序员的支持，随后Linux正式进入如火如荼的发展阶段。
  2. Linux系统的吉祥物名为Tux，是一只呆萌的小企鹅。

    常见的Linux系统版本

    在介绍常见的Linux系统版本之前，首先需要区分Linux系统内核与Linux发行套件系统的不同。
  • Linux内核是操作系统的核心，负责提供基本的操作系统功能。
  • Linux发行套件系统是在内核的基础上，添加了各种实用工具、应用程序和图形界面等组件，形成一个完整的操作系统。

全球的Linux系统大约有近百款，每个系统都有自己的特点与针对性，我这里只列出四种常见的Linux系统。

引用一下刘瑞老师的话: “我们并不是因为开源而喜欢Linux，而是因为Linux系统真的非常优秀，开源精神仅仅是锦上添花而已。”

1. CentOS

  • CentOS是一款基于Red Hat Enterprise Linux（RHEL）的开源操作系统，它具有稳定性高、兼容性强等特点，广泛应用于服务器领域。CentOS全称Community Enterprise Operating System，意为社区企业操作系统。

  • 在CentOS中，使用的包管理器是yum（Yellow dog Updater Modified）yum是一款强大的软件包管理器，它能够方便地安装、升级和删除软件包。

    2. Debian Linux

  • Debian是一款著名的开源操作系统，它以稳定性、安全性和自由度著称。它由Debian Foundation维护。

  • 在Debian中，使用的包管理器是apt（Advanced Package Tool）。apt是一款强大的包管理器，它继承了Debian的稳定性特点。

    3. Ubuntu Linux

  • Ubuntu是一款基于Debian的开源操作系统，由Canonical Ltd.公司支持和发展。它以人性化、易用性和强大的社区支持而闻名。Ubuntu发行版定期发布新版本，并提供广泛的软件包和图形化界面，使得它深受广大用户喜爱。

  • 在Ubuntu Linux中使用的包管理器是apt（Advanced Package Tool），apt具有稳定性、可靠性和自动化等特点，可以满足用户在安装、升级和删除软件包方面的需求。

    4. Arch Linux

  • Arch Linux是一款以简单、轻量级和定制性强著称的开源操作系统。它采用了滚动发布模式，意味着用户可以始终使用最新的软件包。Arch Linux的设计理念强调简洁和自由度，允许用户根据需求自由定制系统。

  • 在Arch Linux中，使用的包管理器是pacman，因它具有易于使用和灵活的特点，所以成为了Arch Linux 社区的标志之一。

> 虽然有这么多Linux系统，但我要说的一点就是——没有最好的系统，只最适合需求的系统。
